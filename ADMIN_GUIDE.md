# Admin Panel - Book Management Guide

## How to Manage Books and Assign Them to Sections

### Step 1: Access Admin Panel
1. Go to `/admin` page in your browser
2. Enter the admin password to login

### Step 2: Upload New Book with Category
1. Fill in the book details (Title, Author, Description)
2. **Select Category** from the dropdown - this determines which section the book appears in:
   - **ရသစာပေ** → Appears in "ရသစာပေများ" section
   - **အောင်မြင်ရေး** → Appears in "အောင်မြင်ရေးစာပေများ" section
   - **ရုပ်ပြ** → Appears in "ရုပ်ပြစာအုပ်များ" section
   - **ဝတ္ထုတို** → Appears in "ဝတ္ထုတိုများ" section
   - **သုတ** → Appears in "သုတစာပေများ" section
   - **ကဗျာ** → Appears in "ကဗျာစာအုပ်များ" section
   - **ဘာသာပြန်** → Appears in "ဘာသာပြန်စာအုပ်များ" section
   - **ဘာသာရေး** → Appears in "ဘာသာရေးစာအုပ်များ" section
3. Upload cover image (optional)
4. Upload PDF file
5. Check "Trending" checkbox if you want it in Trending section
6. Click "Upload Book"

### Step 3: Edit Existing Book Category
1. Scroll down to "Manage Books" section
2. Find the book you want to edit
3. Click "Edit" button
4. Change the Category dropdown to the desired category
5. Click "Save Changes"

### Step 4: Delete a Book
1. Scroll down to "Manage Books" section
2. Find the book you want to delete
3. Click "Delete" button
4. Confirm the deletion

### Step 5: Toggle Trending Status
1. Scroll down to "Manage Books" section
2. Use the toggle switch in the "Trending" column to show/hide books in Trending section

## Category to Section Mapping

| Category Value | Home Page Section |
|---------------|-------------------|
| ရသစာပေ | ရသစာပေများ |
| အောင်မြင်ရေး | အောင်မြင်ရေးစာပေများ |
| ရုပ်ပြ | ရုပ်ပြစာအုပ်များ |
| ဝတ္ထုတို | ဝတ္ထုတိုများ |
| သုတ | သုတစာပေများ |
| ကဗျာ | ကဗျာစာအုပ်များ |
| ဘာသာပြန် | ဘာသာပြန်စာအုပ်များ |
| ဘာသာရေး | ဘာသာရေးစာအုပ်များ |

## Notes
- Books will automatically appear in their corresponding section on the home page
- You can assign a book to Trending section separately using the toggle
- When editing, you only need to upload new files if you want to replace existing ones
